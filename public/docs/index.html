Introduction

Base URL: http://localhost:8004

This documentation aims to provide all the information you need to work with our API.

As you scroll, you'll see code examples for working with the API in different programming languages in the dark area to the right (or as part of the content on mobile). You can switch the language used with the tabs at the top right (or from the nav menu at the top left on mobile).

Authenticating requests

This API is not authenticated.

Endpoints

POST api/register

Example request:
curl --request POST \
    "http://localhost:8004/api/register" \
    --header "Content-Type: application/json" \
    --header "Accept: application/json" \
    --header "Authorization: Bearer " \
    --data "{
    \"first_name\": \"ksnikbesdijpguxi\",
    \"last_name\": \"pxkabevwdfbwfzbqwcqgjvmxkicwezuyymnqkbybhbmqoubmrrxkeuqsxkbrnpcrgydntcutidlffqg\",
    \"phone_number\": \"qui\",
    \"email\": \"arturo35@example.net\",
    \"password\": \"NpKh[AV\"
}"

const url = new URL(
    "http://localhost:8004/api/register"
);

const headers = {
    "Content-Type": "application/json",
    "Accept": "application/json",

};

let body = {
    "first_name": "ksnikbesdijpguxi",

    "email": "arturo35@example.net",

Request      

POST api/transactions

Headers

Content-Type      

Example: application/json

Accept      

Example: application/json

Authorization      

Example: Bearer

Body Parameters

amount   number   

Must be at least 0.01. Example: 29

currency_id_from   string   

The id of an existing record in the currencies table. Example: qui

currency_id_to   string   

The id of an existing record in the currencies table. Example: earum

method   string   

Example: cash_pick_up

Must be one of:
  • mobile_money
  • bank_deposit
  • cash_pick_up
save_recipient   boolean  optional  

Example: false

recipient   object   
first_name   string  optional  

This field is required when method is mobile_money or cash_pick_up. Must not be greater than 255 characters. Example: rja

last_name   string  optional  

This field is required when method is mobile_money or cash_pick_up. Must not be greater than 255 characters. Example: brewvwntbxfopezcqmimmgh

phone_number   string  optional  

This field is required when method is mobile_money. Must not be greater than 15 characters. Example: ase

email   string  optional  

This field is required when method is cash_pick_up. Must be a valid email address. Must not be greater than 255 characters. Example: wilson.borer@example.org

bank_name   string  optional  

This field is required when method is bank_deposit. Must not be greater than 255 characters. Example: vihljxtmiwtldqoj

account_name   string  optional  

This field is required when method is bank_deposit. Must not be greater than 255 characters. Example: kszhziqgrligcb

account_number   string  optional  

This field is required when method is bank_deposit. Must not be greater than 50 characters. Example: ecsqjar
